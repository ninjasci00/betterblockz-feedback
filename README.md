# BetterBlockZ Feedback

This repository is for **BetterBlockZ feedback and issue tracking**.

Please use the appropriate **issue form** when opening an issue:
- [Bug report](https://github.com/ninjasci00/betterblockz-feedback/issues/new?template=bug_report.yml) - report crashes, glitches, or incorrect behavior
- [Feature / Texture suggestion](https://github.com/ninjasci00/betterblockz-feedback/issues/new?template=feature_request.yml) - suggest new textures, variations, or features
- [Question](https://github.com/ninjasci00/betterblockz-feedback/issues/new?template=question.yml) - general questions or clarification
- [General feedback](https://github.com/ninjasci00/betterblockz-feedback/issues/new?template=general_feedback.yml) - share impressions, ideas, or thoughts that don’t fit other categories

For translation contributions, use the dedicated repository:  
→ https://github.com/ninjasci00/betterblockz-community ←
